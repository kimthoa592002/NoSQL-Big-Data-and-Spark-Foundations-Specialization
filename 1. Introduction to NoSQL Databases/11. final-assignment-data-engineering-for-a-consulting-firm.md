***+ Exercise 2: Find the year with the most movies (2016 - 73 movies)***

db.movies.aggregate(\[

{ "$group": { "\_id": "$year", "moviecount": { "$sum": 1 } } },

{ "$sort": { "moviecount": -1 } },

{ "$limit": 1 }

])



***+ Exercise 3: Count movies released after 1999 (99 movies)***

db.movies.countDocuments({ year: { $gt: 1999 } })



***+ Exercise 4: Average votes for movies in 2007 (192.5)***
