## Lab: Keyspace Operations - Coursera Cloud IDE

Environment:

\- Platform: Coursera / Cognitive Class

\- Tool: Cloud IDE (browser-based)

\- Database: Cassandra

\- Access: Cassandra CLI (cqlsh)

---

### 1. Create a keyspace

CREATE KEYSPACE training

WITH replication = {'class':'SimpleStrategy', 'replication\_factor': 3};



***+ To verify, list all keyspaces:***

describe keyspaces



### 2. Describe a keyspace

describe training



### 3. Alter a keyspace

ALTER KEYSPACE training

WITH replication = {'class': 'NetworkTopologyStrategy'};



### 4. Use a keyspace

use training

describe tables



### 5. Drop a keyspace

drop keyspace training;



***+ To verify deletion, switch back to the system keyspace and list all keyspaces:***

use system;

describe keyspaces



### 6. Practice Exercises

***+ Create a keyspace named sales with SimpleStrategy and a replication factor of 1***

CREATE KEYSPACE sales

WITH replication = {'class':'SimpleStrategy', 'replication\_factor': 1};



***+ Alter the replication factor of the sales keyspace to 3***

ALTER KEYSPACE sales

WITH replication = {'class': 'SimpleStrategy', 'replication\_factor': 3};



***+ Drop the keyspace and verify***

drop keyspace sales;

describe keyspaces



***+ Exit the server connection***

exit
