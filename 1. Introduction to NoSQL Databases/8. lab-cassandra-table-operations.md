**Lab: Cassandra Table Operations - Coursera Cloud IDE**



Environment:

\- Platform: Coursera / Cognitive Class

\- Tool: Cloud IDE (browser-based)

\- Database: Cassandra

\- Access: Cassandra CLI (cqlsh)



\--------------------------------------------------



**# 1. Create a keyspace named `training` using SimpleStrategy and a replication factor of 3**

CREATE KEYSPACE training

WITH replication = {'class': 'SimpleStrategy', 'replication\_factor': 3};



use training;



**# 2. Create a table named `movies` in the keyspace training**

use training; 

CREATE TABLE movies( 

movie\_id int PRIMARY KEY, 

movie\_name text, 

year\_of\_release int 

);



***+ Verify the table creation by listing all tables:***

describe tables;



**# 3. Display detailed information about the structure and settings of the movies table**

describe movies;



**# 4. Modify the existing table by adding a new column named genre**

ALTER TABLE movies

ADD genre text;



**# 5. Remove the movies table from the keyspace**

drop table movies;
