## Hadoop MapReduce WordCount - Coursera Cloud IDE
Environment:
- Platform: Coursera / Cognitive Class / Skills Network
- Tool: Cloud IDE (Theia-based)
- Framework: Apache Hadoop 3.3.6
- Mode: Single-node setup
---
### 1. Open a new terminal
Go to Terminal -> New Terminal

### 2. Download Hadoop 3.3.6
curl https://www.google.com/search?q=https://dlcdn.apache.org/hadoop/common/hadoop-3.3.6/hadoop-3.3.6.tar.gz --output hadoop-3.3.6.tar.gz

### 3. Extract the tar file
tar -xvf hadoop-3.3.6.tar.gz

### 4. Navigate to the Hadoop directory
cd hadoop-3.3.6

### 5. Verify Hadoop installation
bin/hadoop

### 6. Download sample data file
curl https://www.google.com/search?q=https://cf-courses-data.s3.us.cloud-object-storage.appdomain.cloud/IBM-BD0225EN-SkillsNetwork/labs/data/data.txt --output data.txt

### 7. Run the WordCount MapReduce application
bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-3.3.6.jar wordcount data.txt output

### 8. Check the output directory
ls output
(Look for part-r-00000 and _SUCCESS)

### 9. View the WordCount results
cat output/part-r-00000

### 10. Lab Practice: Custom WordCount

***+ 10.1 Remove previous data and output***

rm data.txt
rm -rf output

***+ 10.2 Create a new data.txt with the following content***

Italy Venice
Italy Pizza
Pizza Pasta Gelato

***+ 10.3 Run WordCount on the new file***

bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-3.3.6.jar wordcount data.txt output

***+ 10.4 Verify custom results***

cat output/part-r-00000
(Expected output: Gelato 1, Italy 2, Pasta 1, Pizza 2, Venice 1)
