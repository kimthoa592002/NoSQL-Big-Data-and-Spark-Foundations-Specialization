## Submit Apache Spark Application - Coursera Cloud IDE

Environment:
- Platform: Coursera / Cognitive Class / Skills Network
- Tool: Cloud IDE (Theia-based)
- Framework: Apache Spark 3.3.3
- Language: Python (PySpark)
---
### 1. Start Spark Master
***+ Download & Extract Spark: Run the following to set up the Spark environment***
wget https://archive.apache.org/dist/spark/spark-3.3.3/spark-3.3.3-bin-hadoop3.tgz && tar xf spark-3.3.3-bin-hadoop3.tgz && rm -rf spark-3.3.3-bin-hadoop3.tgz

***+ Set Environment Variables***: Define `JAVA_HOME` and `SPARK_HOME`[cite: 12]:
    ```bash
    export JAVA_HOME=/usr/lib/jvm/java-1.11.0-openjdk-amd64
    export SPARK_HOME=/home/project/spark-3.3.3-bin-hadoop3
    ```
    
***+ Configure Master***: Create `spark-defaults.conf` and set executor resources[cite: 12]:
    ```bash
    touch $SPARK_HOME/conf/spark-defaults.conf
    # Add these lines to the file:
    # spark.executor.memory 4g
    # spark.executor.cores 2
    ```
    
***+ Launch Master***: Navigate to `SPARK_HOME` and start the service[cite: 12]:
    ```bash
    cd $SPARK_HOME
    ./sbin/start-master.sh
    ```
    
***+ Verify***: Open the **Spark Master UI** (Port 8080) and **copy the Master URL** (e.g., `spark://theiadocker-yourname:7077`)[cite: 12].

### 2. Start Spark Worker

***+ Open New Terminal***: Open a second terminal window[cite: 12].

***+ Set Variables***: Re-export `JAVA_HOME` and `SPARK_HOME`[cite: 12].

***+ Launch Worker***: Connect the worker to the Master using the URL copied earlier[cite: 12]:

    ```bash
    cd $SPARK_HOME
    ./sbin/start-worker.sh spark://theiadocker-yourname:7077 --cores 1 --memory 1g
    ```
    
***+ Verify***: Refresh the Master UI to see **1 Alive Worker** registered[cite: 12].

### 3. Create and Submit Application

***+ Create Python Script***: In a new terminal, create `submit.py`[cite: 12]:
    ```bash
    touch submit.py
    ```
    
***+ Add Code***: Paste the PySpark code into `submit.py`. **Crucial**: Replace `yourname` in the `.master()` configuration with your actual Spark Master URL[cite: 12]:

    ```python
    import findspark
    findspark.init()
    from pyspark.sql import SparkSession
    # ... (refer to source code for full imports/schema)
    spark = SparkSession.builder \
        .master('spark://theiadocker-yourname:7077') \
        .config('spark.executor.memory', '512m') \
        .getOrCreate()
    # ... (DataFrame creation logic)
    df.show()
    ```
    
***+ Install Dependencies***: Install the `findspark` library[cite: 12]:
    ```bash
    pip3 install findspark
    ```
    
***+ Execute***: Run the script[cite: 12]:
    ```bash
    python3 submit.py
    ```

### 4. Monitoring

***+ Master UI (Port 8080)***: Check the **Completed Applications** section to see your `pyspark-shell` task with a "FINISHED" state[cite: 12].

***+ Worker UI (Port 8081)***: View specific details about the worker currently running or that finished the task[cite: 12].
