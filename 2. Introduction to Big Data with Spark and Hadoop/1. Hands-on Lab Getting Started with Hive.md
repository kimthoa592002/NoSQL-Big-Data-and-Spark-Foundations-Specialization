## Getting started with Hive - Coursera Cloud IDE

Environment:

\- Platform: Coursera / Cognitive Class / Skills Network

\- Tool: Cloud IDE (Theia-based)

\- Service: Docker, Apache Hive

\- Access: Beeline CLI

---

### 1. Prepare data directory

mkdir /home/project/data

cd /home/project/data



### 2. Download sample CSV file

wget https://www.google.com/search?q=https://cf-courses-data.s3.us.cloud-object-storage.appdomain.cloud/IBM-BD0225EN-SkillsNetwork/data/emp.csv



### 3. Pull Hive Docker image

docker pull apache/hive:4.0.0-alpha-1



### 4. Start Hive server container

docker run -d -p 10000:10000 -p 10002:10002 --env SERVICE\_NAME=hiveserver2 -v /home/project/data:/hive\_custom\_data --name myhiveserver apache/hive:4.0.0-alpha-1



### 5. Connect to Hive using Beeline

docker exec -it myhiveserver beeline -u 'jdbc:hive2://localhost:10000/'



### 6. Create Employee table

CREATE TABLE Employee(emp\_id string, emp\_name string, salary int) ROW FORMAT DELIMITED FIELDS TERMINATED BY ',';



### 7. List tables

show tables;


### 8. Load data from CSV into Hive table

LOAD DATA INPATH '/hive\_custom\_data/emp.csv' INTO TABLE Employee;



### 9. Query all data from the table

SELECT \* FROM Employee;



### 10. Disconnect from Beeline

Ctrl+D (or exit)
